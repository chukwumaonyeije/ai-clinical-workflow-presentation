### Modify Slides

Edit the content within the `<div class="slides">` section in `index.html`.

Each slide is wrapped in `<section>` tags:

```html
<section>
    <h2>Your Slide Title</h2>
    <p>Your content here</p>
</section>
```

For vertical slides (sub-slides):

```html
<section>
    <section>Main slide</section>
    <section>Sub-slide 1</section>
    <section>Sub-slide 2</section>
</section>
```
